# AGENTS.md

## Repo Rules For Codex

1. Use `pnpm` only.

- Use `pnpm` for installs, dependency changes, and script execution.
- Do not use `npm` commands.
- Keep `pnpm-lock.yaml` as the source of truth for lockfile updates.

2. Use English only in the codebase and commits.

- Write code comments, documentation, and commit messages in English.
- Do not introduce non-English text unless explicitly requested.
